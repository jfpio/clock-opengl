# GKOM Clock OpenGL project 
## Usage
Open in Visual Studio and run project

## Implemented functionality 
1.  The clock with moving clock hands
2.  Textured clock and area
3.  Skybox
4.  Lighting
5.  Moving camera


## Sterowanie
* X Y - speed of clock hands
* V C - light intensity
* WASD, mouse - camera move
* Esc exit
